# tech-scout

このプロジェクトの目的は、新しい技術、SaaS、ライブラリなどを検証し、自分自身の技術的な引き出し（Technology Radar）を構築することです。

## 評価基準（Tech Radar）

各項目（検証対象）は、以下の基準で評価・分類されます。

- **Adopt**: 自信を持って提案できる、プロダクション導入を推奨する技術
- **Trial**: PoCで手触りを確認済み、次のプロジェクトで積極的に使いたい技術
- **Assess**: 気になっているが、まだ検証未完了の技術
- **Hold**: 過去に使ったが非推奨、またはより優れた代替手段が存在する技術
